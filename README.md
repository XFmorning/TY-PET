# TY AI - 天依形象桌面宠物

一个基于 Electron 的透明 WebM 桌面宠物，支持多种交互动画、键盘检测工作循环、拖放移动。

## 功能

- **透明渲染** — Canvas 绘制 WebM 视频，支持透明通道，完美融入桌面
- **多动画状态** — 待机、打招呼、调皮、蹲下、画圈圈、跳舞、消息来了
- **工作循环** — 键盘检测自动触发打字动画（开始打字 → 工作中 → 结束）
- **拖放移动** — 拖拽顶部圆点移动宠物位置，带拎起来/放下去动画
- **对话气泡** — 触发动画时显示可配置的对话文字
- **系统托盘** — 最小化到托盘，右键菜单快速操作
- **管理面板** — 动画预览、对话编辑、置顶/自启/随机动作设置

## 技术栈

- **Electron 42** — 桌面框架
- **WebM (VP9 alpha)** — 透明视频格式
- **Canvas 2D** — 视频帧渲染
- **Koffi** — FFI 键盘监听
- **electron-builder** — 打包分发

## 目录结构

```
├── main/                   # 主进程
│   ├── index.js            # 入口、窗口管理、IPC
│   ├── tray.js             # 系统托盘
│   ├── state-machine.js    # 状态机
│   ├── animations.js       # 动画映射
│   ├── settings.js         # 设置持久化
│   ├── keyboard-monitor.js # 键盘监听
│   ├── preload-pet.js      # 宠物窗口预加载
│   └── preload-settings.js # 设置窗口预加载
├── renderer/
│   ├── pet/                # 宠物窗口（Canvas + 动画）
│   ├── settings/           # 管理面板
│   ├── speech-bubble/      # 对话气泡窗口（备用）
│   └── ctx-menu/           # 右键菜单窗口
├── animations/             # 动画 WebM 文件
├── assets/                 # 图标等资源
└── package.json
```

## 开发

```bash
npm install
npm start        # 启动应用
npm run dist     # 打包分发
```

## 动画列表

| 动画 | 触发方式 | 循环 |
|------|---------|------|
| 待机 | 默认 | 是 |
| 打招呼 | 单击宠物 | 否 |
| 调皮 | 快速连击 | 否 |
| 蹲下/画圈圈 | 随机动作 | 否 |
| 打字工作 | 键盘检测 | 序列 |
| 跳舞 | 右键菜单 | 否 |
| 消息来了 | 右键菜单 | 否 |
